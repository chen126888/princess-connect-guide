# 公主連結攻略網站

一個基於 React + Node.js 的公主連結遊戲攻略網站。

## 專案結構

- `frontend/` - React + TypeScript + Tailwind CSS 前端
- `backend/` - Node.js + Express + PostgreSQL 後端
- `database/` - 資料庫遷移和種子資料
- `shared/` - 前後端共用的類型定義
- `docs/` - 專案文檔

## 開發環境設定

請參考 docs/SETUP.md

## 功能特色

- 角色資料管理
- 6星角色優先順序指南
- 商店兌換建議
- PvP 隊伍配置
- 戰隊戰攻略
- 深域+戰鬥試煉指南
